PDF マスキングツール

AI送信前に個人情報をマスキング

使い方ガイド

PDFを読み込む

ファイルをドラッグ＆ドロップするか、「ファイルを選択」ボタンからPDFを開きます。

マスキングツール

墨消し ドラッグで四角形の墨消しを作成。連続で何個でも描けます。
既存マスクをクリック → 自動で選択モードに切替
太消し 太いペンで線を引いてマスク。
Shift + ドラッグ → 完全な水平・垂直ライン
細消し 細いペンで線を引いてマスク。
Shift + ドラッグ → 水平・垂直ライン
自由消し フリーハンドで塗りつぶしてマスク。不規則な形状に対応。
全消し ページ全体をマスク。再度押すと解除。
選択モードに切り替わり、ハンドルで範囲調整可能
選択 マスクをクリックで選択。
ドラッグ → 移動 ／ 角のハンドル → リサイズ

編集操作

🎨 色カラーピッカーでマスクの色を変更
↩ 元に戻す直前のマスクを取り消し
🗑 クリア現在ページのマスクを全て削除
Del選択中のマスクを削除（Backspace も可）

ズーム操作

＋ −拡大 / 縮小（50%〜400%）
📐 フィット押すたびに切替：縦フィット → 横フィット → 100%
🤏 ピンチスマホで2本指ズームイン/アウト

ページ移動

◀ ▶ ボタンで前後のページに移動。各ページ個別にマスクが保持されます。

保存

「📥 PDFダウンロード」でマスク済みPDFを保存。

  • ファイル名: 元のファイル名_masked.pdf
  • 元のテキスト情報は削除 → 画像化PDFとして出力（テキスト抽出不可）
  • スマホではOSの共有メニューが表示されます

スマホでの操作

👆 1本指描画 / 選択 / スクロール
🤏 2本指ピンチでズームイン/アウト
☝️ マスク上タッチ → 選択 → ドラッグで移動/リサイズ

セキュリティについて

このツールは完全にブラウザ内で動作します。PDFファイルがサーバーに送信されることはありません。
